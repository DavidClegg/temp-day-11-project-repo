My first difficult project

My journey for web development started with FreeCodeCamp. FreeCodeCamp is a teaching tool for many web technologies. It was an uphill struggle at first, but the self-paced nature of the content helped me learn the basic fundamentals of web development.

From there I began trying to create simple tools to help with my daily life. My first project was a working Note Taker. Initially I started out with a simple list and an input box in HTML.

<ul> <li>Item 1</li> <li>Item 2</li> <li>Item 3</li> </ul> <input type="text"> <button click="additem">Add Item</button>
  • Item 1
  • Item 2
  • Item 3

This was fun to look at, once I wrote the code to add items to the list:

function additem(){ let input = document.querySelector("input"); list.innerHTML += `<li>${input.value}</li>` }

But losing my shopping list when I refresh the page was frustrating.

I took to Google and started furiously typing in every solution to the problem I could think of. "Other websites store data; How hard could it be?!"

Eventually I stumbled upon the localstorage API. By creating a list of notes as a JSON object, I was able to save between sessions locally. All it took was a couple of functions (one to save and one to load), splitting the data string, and my app would do everything I needed.

function save(){ localstorage.setItem('Notes', list) } function load(){ let data = localstorage.getItem('Notes'); list = data.split(","); }

No longer would my shopping list be lost to the aether after refreshing my page!
